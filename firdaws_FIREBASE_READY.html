🔥 المزامنة السحابية (Firebase)

✅ Firebase مُفعَّل — المزامنة التلقائية تعمل بين جميع الأجهزة
المشروع: firdaws-b9c9a
📖 كيفية إعداد Firebase
  1. اذهب إلى console.firebase.google.com
  2. اضغط "Add project" وأنشئ مشروعاً جديداً (مجاني)
  3. من القائمة اليسرى، اختر Build → Firestore Database
  4. اضغط "Create database" → اختر Start in test mode → Enable
  5. ارجع لـ Project Overview → اضغط أيقونة Web (</>)
  6. أدخل اسم التطبيق واضغط Register
  7. انسخ كائن firebaseConfig (يبدأ بـ apiKey)
  8. افتح هذا الملف في محرر نصي → ابحث عن YOUR_API_KEY_HERE
  9. الصق بيانات Firebase مكان القيم الافتراضية
  10. غيّر FIREBASE_ENABLED = false إلى FIREBASE_ENABLED = true
  11. احفظ الملف وارفعه على Netlify
  12. المزامنة ستبدأ تلقائياً! ستظهر أيقونة 🔥 في أسفل الشاشة يسار
💡 كيف تعمل المزامنة:
• عند إضافة أي بيانات (مريض، عملية، مستخدم)، تُحفظ محلياً ثم تُرفع لـ Firebase تلقائياً بعد 1.5 ثانية.
• عند حدوث تغيير في أي جهاز آخر، يصل التحديث لجميع الأجهزة المتصلة فوراً دون الحاجة للتحديث اليدوي.
• الأيقونة 🔥 في الأسفل تُظهر حالة المزامنة (⬆️ رفع / ⬇️ تحميل / ✅ متصل).
